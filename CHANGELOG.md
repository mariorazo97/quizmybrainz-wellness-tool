# Changelog

All notable changes to the Body & Mind Detective project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-29

### Added
- Initial release of Body & Mind Detective
- Student "Body Detective" Quiz with 4 interoception questions
- Teacher/Project Leader High-Functioning Burnout Assessment with 5 evidence-based questions
- Emoji-based interface for student responses
- Yes/No with intensity sliders for teacher assessment
- Personalized result pages with actionable strategies
- Breathing bubble animation (8-second cycle) for students
- Recovery Toolkit with 3 expandable strategy cards for teachers:
  - Vagus Nerve Reset protocol
  - Delay Script for boundary setting
  - Eisenhower Matrix for prioritization
- Glassmorphism design system with golden color palette
- Smooth page transitions (0.4s ease-in-out)
- Responsive mobile-first design
- Persistent home button navigation
- Progress tracking for both quiz types
- External links to peer-reviewed scientific resources
- MIT License
- Comprehensive README with deployment instructions
- Contributing guidelines
- GitHub Actions workflow for automatic deployment

### Design Features
- Crimson Pro + DM Sans typography combination
- Warm cream background (#F9F7F2) for reduced eye strain
- Golden primary color (#D4AF37) with red-orange accents (#FF4500)
- Radial gradient atmospheric effects
- CSS-only animations (no JavaScript animation libraries)
- High contrast ratios for accessibility (WCAG 2.1 AA compliant)

### Technical
- Single-file HTML application (no build process required)
- Vanilla JavaScript (zero dependencies)
- ~20KB uncompressed file size
- Compatible with all modern browsers
- Semantic HTML5 structure
- Client-side state management
